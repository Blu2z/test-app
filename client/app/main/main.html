Создать Single Page Application используя MEAN стэк, тема — каталог товаров.

Требования:

  1. Приложение должно иметь 3 раздела:
    • Каталог — список названий товаров. Данные беруться с сервера и выводяться динамически с помощью Angular.js
    • Раздел 2
    • Раздел 3
    Раздел 2 и 3 — произвольные разделы в которых может содержаться любая информация. Приложение содержит 3 темплейта и соответственно роутинг должен производиться средствами Angular.js
  2. Создать сервер построенный на Node.js с использованием Express.js.
    На стороне сервера реализовать 2 функции:
    • добавление записи в БД
    • вывод всех записей
  3. База данных — MongoDB Каждая запись содержит 2 поля:
    • id — идентификатор записи
    • name — название товара
